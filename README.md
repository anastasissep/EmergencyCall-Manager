# EmergencyCall-Manager
A Java-based prototype for managing emergency service calls


EmergencyCall Manager is a Java-based project that simulates the core functionality of an emergency call handling system.

The system is designed to record and manage calls made to emergency services such as the police, fire department, and ambulance. Each call contains the caller’s information, a short description of the incident, the selected emergency services, the call status, and the time it was created.

The project also includes basic user management, file-based data storage, and a graphical user interface.

## Features

- Emergency call registration
- Support for multiple emergency services per incident
- User roles such as Citizen and Admin
- Call status management
- Basic graphical user interface
- CSV file storage for users and calls

## Project Structure

- `src/model` – core classes and data models
- `src/controller` – call management and file handling logic
- `src/gui` – graphical user interface components
- `src/main` – main class used to launch the application
- `data` – CSV files used for storing application data

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- NetBeans
- CSV file handling

## How to Run

To run the project correctly, you must first download the entire project folder from GitHub.

After downloading it, open the project in NetBeans and make sure that all classes and folders are included together in the same project structure.

### Steps
1. Download the full project folder.
2. Open NetBeans.
3. Import or open the downloaded folder as a project.
4. Make sure all classes are loaded together in NetBeans.
5. Run the project from the main class.

## Important Note

This project is intended to run as a complete project and not as separate individual `.java` files.

For this reason, you should first download the whole folder and then place/import all classes together in NetBeans before running the application.

## Purpose

This project can serve as a foundation for an emergency services coordination application, helping improve communication and response management between emergency services.
